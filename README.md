# Django Task Manager

A simple task manager web application built with Django.  
Users can create tasks for a specific date, mark them as complete, update them, and delete them.

## Overview

This project is a basic CRUD web application created to practice Django fundamentals:
- models
- forms
- views
- URL routing
- template rendering
- SQLite database integration

## Features

- create a task
- assign a custom date and time
- mark task as complete
- edit existing tasks
- delete tasks
- display tasks in a list view

## Tech Stack

- Python
- Django
- HTML templates
- SQLite

## Project Structure

- `planner/` — Django project configuration
- `tasks/` — main application logic
- `templates/` — HTML templates
- `manage.py` — Django management entry point
